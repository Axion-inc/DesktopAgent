Technical Details

How Planner L1 Works:

  1. Intent Analysis: Uses regex patterns to identify automation goals
  2. Entity Extraction: Identifies file types, actions, and parameters
  3. Template Selection: Chooses appropriate DSL template based on intent
  4. Plan Generation: Fills template with extracted entities
  5. Confidence Scoring: Provides confidence level for generated plan

Supported Patterns:

  • File operations (find, move, organize)
  • PDF processing (merge, extract)
  • Web form automation
  • Email composition and sending
  • CSV data processing
